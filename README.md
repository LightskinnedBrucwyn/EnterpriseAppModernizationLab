# Enterprise Application Modernization Lab

This repository documents hands-on practice for enterprise application modernization, Microsoft development, databases, low-code migration, and secure government software environments.

## Purpose

This lab is designed to build practical skills in:

- C Sharp programming
- Object-oriented programming
- Structured Query Language
- Relational database design
- Dataverse concepts
- Power Pages migration
- Blazor web applications
- Modern .NET development
- .NET Framework to modern .NET migration concepts
- Azure Virtual Desktop fundamentals
- Risk Management Framework awareness
- Security Technical Implementation Guides awareness

## Current Focus

The first learning track focuses on building a foundation in C Sharp, object-oriented programming, Structured Query Language, and relational database concepts.

## Lab Structure

| Folder | Purpose |
|---|---|
| 01-CSharp-Object-Oriented-Programming | C Sharp fundamentals and object-oriented programming practice |
| 02-Structured-Query-Language | Query writing, updates, joins, and data manipulation |
| 03-Relational-Database-Design | Tables, keys, relationships, and normalization |
| 04-Dataverse-Concepts | Tables, columns, choices, lookups, and relationships |
| 05-Power-Pages-Migration | Power Pages concepts and migration notes |
| 06-Blazor-Web-Applications | Blazor practice projects |
| 07-DotNet-Modernization | Legacy .NET Framework to modern .NET notes and examples |
| 08-Azure-Virtual-Desktop | Azure Virtual Desktop concepts and study notes |
| 09-Security-Compliance-Basics | Risk Management Framework and Security Technical Implementation Guides notes |
