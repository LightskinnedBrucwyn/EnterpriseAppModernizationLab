# Projects

## Bat Household Hub

`BatHouseholdHub` is a private, home-hosted Blazor dashboard for shared finances,
credit utilization, recipes, weekly meal planning, and grocery lists.

Run it locally with:

```powershell
dotnet run --project .\BatHouseholdHub\BatHouseholdHub.csproj
```

Hands-on projects that combine C Sharp, Structured Query Language, Dataverse concepts, Blazor, and modernization practice.
